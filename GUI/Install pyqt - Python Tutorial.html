Install pyqt

PyQt is often not installed by default. The PyQt module can be used to create desktop applications with Python. In this article youll learn how to install the PyQt module.

Desktop applications made with PyQt are cross platform, they will work on Microsoft Windows, Apple Mac OS X and Linux computers (including Raspberry Pi).
